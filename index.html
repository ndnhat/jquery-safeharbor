jquery.safeHarbor

Safe Harbor is a jQuery plugin that creates a privacy policy notice and helps your web application conform with the US-EU Safe Harbor. Optionally, Safe Harbor can be paired with jquery-cookie to memorize the user's acknowledgement and not show the notice in subsequent visits.

Getting Started

Download the production version or the development version.

In your web page:

<script src="jquery.js"></script>
<script src="jquery.cookie.js"></script> <!-- Optional -->
<script src="jquery.safeharbor.js"></script>
<script>
jQuery(function($) {
  $.safeHarbor({text: 'This is our <a href="privacy.html">Privacy policy</a>'}); 
});
</script>
